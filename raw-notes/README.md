# Raw Notes

Place your raw notes here for conversion to web format.

## Supported Formats

- **LaTeX** (`.tex`) - Mathematical documents
- **Markdown** (`.md`) - Text notes with formatting
- **PDF** (`.pdf`) - Handwritten or scanned notes

## Usage

1. Add your raw note files to this directory
2. Ask Claude to convert them into web notes
3. The converted notes will be added to `/notes/` with MathJax and Plotly support
