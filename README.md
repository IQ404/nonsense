# Personal Website

**Live Site:** https://iq404.github.io/nonsense/

A clean, modern personal portfolio website built with HTML and CSS.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Hero section with call-to-action buttons
- About section with skills display
- Projects showcase grid
- Contact section with social links
- Modern, minimal aesthetic

## Customization

To personalize this website, update the following in `index.html`:

1. **Your Name** - Replace "Your Name" throughout the file
2. **Tagline** - Update the hero subtitle (Developer | Designer | Creator)
3. **About Text** - Add your own bio and background
4. **Skills** - List your technologies and tools
5. **Projects** - Add your own projects with descriptions and links
6. **Contact** - Update email and social media links

## Deployment

This site is configured for GitHub Pages. Simply push to the `main` branch and enable GitHub Pages in your repository settings.
